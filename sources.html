Soureces

Some datasets and codes are shared with you.

Here are some sources including hyperspectral datasets and codes, some of which are collected from the Internet. I hope these sources are helpful to your work.

I. Datasets

  1. ABU datasets
    • description: This dataset is provided by Prof. Xudong Kang and is mainly used for hyperspectral anomaly detection.
    • website: http://xudongkang.weebly.com/data-sets.html
  2. Salinas datasets
    • description: This dataset is provided by Mr. Donglei Ma and Dr. Zengfu Hou, and is mainly used for hyperspectral anomaly detection. If you use this dataset, please cite the following papers.
      • [1] Anomaly detection in hyperspectral imagery based on low-rank representation incorporating a spatial constraint[J]. Remote Sensing, 2019, 11(13): 1578.
      [2] Anomaly Detection for Hyperspectral Imagery Based on the Regularized Subspace Method and Collaborative Representation[J]. Remote Sensing, 2019, 11(11): 1318.
    • website: Google drive or Baidu Netdisk, Code: g7wu
  3. HyMap Target/Anomaly Detection Blind Test dataset
    • description: The Target Detection Blind Test project aims to provide a standard hyperspectral dataset to the remote sensing community. Additionally, the project will score the ability of target finding algorithms used on the dataset provided. These services are provided for free — after registering, anyone can access this project's data and evaluation services.
    • website: http://dirsapps.cis.rit.edu/blindtest
  4. Updating...

II. Codes

  1. Morphological feature extraction program (Max-tree and Min-tree)
    • description: The morphological feature extraction in max-tree/min-tree was conducted in VISUAL STUDIO 2017 using C++ language, which provied by prof. Lu Li, and modified and encapsulated by Dr. Zengfu Hou. We have released these codes. For more details, please refer to:
      [1] Hyperspectral Change Detection Based on Multiple Morphological Profiles[J]. IEEE Transactions on Geoscience and Remote Sensing, 2021, doi:10.1109/TGRS.2021.3090802.
    • C++ Codes (Executable program): Download

B. Anomaly Detection

  1. Hyperspectral Anomaly Detection: LSADCRIDW/LSUNRSORAD
    • description: This is the matlab code for hyperspectral anomaly detection. For more details, please refer to the paper:
      [1] Anomaly Detection for Hyperspectral Imagery Based on the Regularized Subspace Method and Collaborative Representation, Remote Sensing, 2019, 11(11): 1318.
    • Matlab Codes: Download
  2. Hyperspectral Anomaly Detection: SLW_LRRSTO/MLW_LRRSTO
    • description: This is the matlab code for hyperspectral anomaly detection. For more details, please refer to the paper:
      [1] Anomaly Detection in Hyperspectral Imagery Based on Low-Rank Representation Incorporating a Spatial Constraint, Remote Sensing, 2019, 11(13): 1578.
    • Matlab Codes: Download
  3. Hyperspectral Anomaly Detection: 2S-GLRT
    • description: This is the matlab code for hyperspectral anomaly detection. For more details, please refer to:
      [1] Multipixel Anomaly Detection With an Unknown Pattern for Hyperspectral Imagery[J]. IEEE Transactions on Neural Networks and Learning Systems, 2021, doi: 10.1109/TNNLS.2021.3071026.
    • Matlab Codes: Download
  4. Hyperspectral Anomaly Detection: CRDBPSW
    • description: This is the matlab code for hyperspectral anomaly detection. For more details, please refer to:
      [1] Collaborative Representation with Background Purification and Saliency Weight for Hyperspectral Anomaly Detection[J]. Science China Information Sciences, 2022 Jan, 65(1):1-12.
    • Matlab Codes: Download

C. Change Detection

  1. Hyperspectral Change Detection: maxtreeCD/mintreeCD/SALA
    • description: This is the matlab and C++ codes for hyperspectral change detection. For more details, please refer to:
      [1] Hyperspectral Change Detection Based on Multiple Morphological Profiles[J]. IEEE Transactions on Geoscience and Remote Sensing, 2021, doi:10.1109/TGRS.2021.3090802.
    • C++/Matlab Codes: Download
  2. Hyperspectral Change Detection: TDRD
    • description: This is the matlab and C++ codes for hyperspectral change detection. For more details, please refer to:
      [1] Three-Order Tucker Decomposition and Reconstruction Detector for Unsupervised Hyperspectral Change Detection[J]. IEEE Journal of Selected Topics in Applied Earth Observations and Remote Sensing, 2021, doi: 10.1109/JSTARS.2021.3088438.
    • Matlab Codes: Download
  3. Hyperspectral Change Detection: PTCD
    • description: This is the matlab and C++ codes for hyperspectral change detection. For more details, please refer to:
      [1] A Patch Tensor-Based Change Detection for Hyperpectral Images[C]. IEEE International Geoscience and Remote Sensing Symposium, 2021, pp.4328-4331.
    • Matlab Codes: Download
